## Description

## Features

    * RESTful API for managing Todo items.
    * Basic user registration/authentication
    * Uses NestJS, a powerful and scalable Node.js framework.
    * Utilizes Prisma ORM to simplify database operations and interactions.
    * Separate development and test databases running in Docker containers built from PostgreSQL images.

## Getting Started

Follow these steps to set up and run the application:

    1. Install dependencies using `npm install`.
    2. Start the development database container using `npm run db:dev:up`.
    3. Run the application using `npm run start:dev`.
    4. Access the API at `http://localhost:5000`.

## Manage the database
```bash
# Start development database
$ npm run db:dev:up

# Stop development database
$ npm run db:dev:down

# Restart development database (Stops the running db container, removes the docker volume of the db and starts up a new clean one)
$ npm run db:dev:restart

# There are similar commands for the test db, but those are just used for the purpose of tests
```

## Running migrations

```bash
# Create a migration from changes in Prisma schema, apply it to the database, trigger generators (e.g. Prisma Client)
$ prisma migrate dev

# Apply pending migrations to the database in production/staging
$ prisma migrate deploy
```

## Test

```bash
# e2e tests
$ npm run test:e2e
```
